Sistema operativo:
Es el software principal del computador. Permite administrar los recursos y ejecutar programas como Node.js.

Node.js:
Es un entorno que permite ejecutar código JavaScript fuera del navegador. Se usa para crear servidores y aplicaciones web.

Aplicación server.js:
Es el archivo que contiene el código del servidor. Define las rutas y respuestas que recibirán los usuarios.

Puerto:
Es el canal de comunicación que utiliza el servidor para recibir solicitudes. En este ejercicio se usa el puerto 3000.

Logs:
Son registros de eventos y actividades del servidor. Ayudan a monitorear el funcionamiento y detectar errores.
